# CSL Vanguard — Pioneering Web Solutions

CSL Vanguard is a modern, high-performance web development agency website built with Next.js 15+, TypeScript, and Tailwind CSS 4. It features a sleek, professional design with smooth animations and a focus on conversion.

## 🚀 Key Features

- **Modern Tech Stack**: Built with Next.js 15, TypeScript, and Tailwind CSS 4.
- **Fluid Animations**: Powered by Framer Motion for a premium user experience.
- **Responsive Design**: Fully optimized for all screen sizes.
- **Interactive Sections**: Dynamic hero, services grid, tech stack marquee, and process timeline.
- **Blog System**: A clean, readable blog with category filtering and dynamic routing.
- **Contact Form**: Integrated contact form with validation and success states.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org) (App Router)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev)
- **Language**: [TypeScript](https://www.typescriptlang.org)

## 📁 Project Structure

```text
src/
├── app/            # Next.js App Router pages and layouts
├── components/     # Reusable UI components
├── lib/            # Utility functions and data
└── styles/         # Global styles (globals.css)
```

## 🏗️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
